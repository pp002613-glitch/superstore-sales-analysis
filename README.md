# 🛒 Superstore Sales Analysis

Exploratory Data Analysis on 9,994 rows of retail sales data
to uncover business insights for decision making.

## 📊 Key Findings
- West region leads with $725K sales — 31% of total revenue
- Technology is most profitable category ($145K profit)
- Furniture barely profitable at $18K — needs investigation
- Sales grew 51% from 2014 to 2017
- Discounts above 40% consistently cause losses (correlation: -0.22)

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
